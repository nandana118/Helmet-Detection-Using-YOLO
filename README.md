# Helmet-Detection-Using-YOLO
Developed a real-time helmet detection system using the Ultralytics YOLO framework and Python to identify helmeted and non-helmeted riders from images and videos. The project involved downloading and preprocessing a helmet detection dataset from Kaggle, training a custom YOLO model using transfer learning, and evaluating its performance for object detection tasks. Implemented image and video inference pipelines to detect helmets with confidence scores and visualize results using bounding boxes. Utilized Google Colab for model training and experimentation, OpenCV for image/video processing, and YOLO11 for high-speed object detection. The trained model was capable of processing multiple frames efficiently and generating annotated outputs for safety monitoring and traffic surveillance applications.
Technologies Used: Python, YOLO (Ultralytics), OpenCV, Google Colab, Kaggle Dataset, Computer Vision, Deep Learning.
